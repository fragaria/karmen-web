Multiple 3D printers controller

Karmen is an open-source solution for multi-printer management. It allows one to control a bunch of printers remotely, features an efficient management of print queues and offers smooth and user-friendly web interface.

  • Attachable control module for each printer

  • Realtime printer webcam stream

  • Simple print queues management
